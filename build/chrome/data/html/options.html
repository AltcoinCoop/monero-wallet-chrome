Thank you for installing Monero Wallet for Chrome!

This page is the configuration page for the extension. Here you will need to set which port the extension can use to talk to simplewallet. You can return to this page in the future from the "Settings" button in the extension menu.

Getting Started

In order to use this extension, simplewallet must be running in RPC mode on the same computer as your Chrome/Chromium browser. For help on setting this up, visit our Getting Started guide.
