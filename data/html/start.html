Monero Wallet

Thanks for using Monero Wallet for Chrome™!

This wallet aims to make Monero's official simplewallet easy to use, combining a visually appealing interface with the functionality of simplewallet. In fact, this wallet is not actually a wallet at all, but rather a layer on top of simplewallet to allow you to use Monero's official command line wallet in an easy and fun.

Getting Started with Windows

Note: This guide assumes that you are familiar with simplewallet, and just need help connecting it to your wallet extension. If you are brand new to Monero and simplewallet, we recommend reading a guide to using simplewallet on getmonero.org.

Initialize a new wallet if not done already

To run simplewallet in rpc mode, you first need to have an initialized and synced wallet. If you do not have this yet, please do so now.

Did you know? It is possible to use Monero's simplewallet as a "lightweight" wallet, connected to a remote node. For example, if you wanted to connect to moneroclub.com's public node, you can add the following flag when opening simplewallet:

simplewallet.exe --daemon-address node.moneroclub.com:8880

Running a wallet in rpc mode

To use this extension, you must launch simplewallet in rpc mode. This requires that you use an extra two flags as part of your launch script, --rpc-bind-ip and --rpc-bind-port. Typically, you want to bind to your localhost IP, 127.0.0.1, and to port 18082. For example:

simplewallet.exe --wallet-file <wallet name> --password <wallet password> --rpc-bind-ip 127.0.0.1 --rpc-bind-port 18082

For simplewallet to connect to your extension, you will need to tell the extension which port to listen on. If you used port 18082, you are ready to go. If not, open the extension's Settings page and set the correct port.

After starting simplewallet in rpc mode, check your chrome extension to see if it is connected online. Click the extension icon in the upper right of your browser to open the extension popup. It may take a few seconds for your extension to fully connect to simplewallet.

Note: When using rpc mode, simplewallet will not respond to terminal commands because interactive mode is disabled. In addition, you cannot close the terminal without closing simplewallet.
Instead, we recommend minimizing the command prompt.

Getting Started with OS X

Installation and Setup

To make launching simplewallet as simple and straightforward as possible, we have created an easy-to-use launcher script for simplewallet. Follow these steps to get your script setup:

  • Open the Finder and navigate to Applications.
  • Make a new folder in Applications named "MoneroWallet".
    Note: it is important that you do not use a different folder name, as our script requires that things are installed exactly in this location.
  • Download the official Monero binaries from getmonero.org for OS X.
    Unzip the downloaded folder and copy the "simplewallet" program into your new "MoneroWallet" directory.
  • Download "Launch Script - OS X.zip" from the latest release on our GitHub page.
    Copy the downloaded script "MoneroWallet.command" into your "MoneroWallet" folder.

If you have followed the steps above correctly, you are almost ready to make a new wallet to use with the Monero Wallet extension. However, let's first talk about the difference between connecting your wallet to your own Monero node and to a remote node.

Before Continuing:
Our launcher script gives the option to connect your wallet to a "remote node", hosted by a member of the Monero community over the internet. It is safe to use such nodes from a security standpoint; node operators do not and cannot know your wallet's secret keys. However, using a remote node does lose some privacy, as the remote node operator can see some of your activity that would normally be hidden. If you want full privacy, you should run your own Monero node. Running your own node is not covered here, but help is available on getmonero.org.

Setting up a new wallet

The first thing the launcher script does is help set up a new wallet with simplewallet. This has to be done via the command line, but we have tried to make this setup as easy as possible. After this initial setup, you should not need the command line wallet again.

  • Double-click "MoneroWallet.command".
    The script will launch in a new Terminal window and display a message. Follow the dialog, choosing a password and other options.
    Unless you have your own Monero node running or want to specify a specific node, choose the "remote" option when prompted.
  • Once you have chosen all of your settings, simplewallet will generate a new wallet for you and tell you a secret 25 word "seed". Write down your seed and keep it safe and private. This seed is all you ever need to restore a corrupt or lost wallet.
  • Simplewallet should have told you to type "refresh" to get your wallet going. Do that now.
    Your wallet will begin to scan Monero's blockchain to bring it up to date. This will take some time.

Once it is done refreshing, simplewallet will tell you it is up to date and display a balance. Close the wallet by typing "exit".

Connecting your extension to simplewallet

Now that you have created a new wallet, the launcher script will work slightly differently. Instead of opening simplewallet in the command line, the script will launch simplewallet in the background on your computer. This allows your wallet extension to talk with simplewallet and send and receive transactions. Any time you want to use your wallet, the script will need to be running.

  • To run the script, double-click the "MoneroWallet.command" file again.
    This time, the script will ask you to enter the password you chose before and pick a node.
    Then it will launch simplewallet in rpc mode, and give you some feedback.
    Once finished, you can safely exit the script by typing "exit" or hitting the "x" button in the top left.
  • Now check your chrome extension to see if it is connected online!
    Click the extension icon in the upper right of your browser to open the extension popup.

You can turn off your wallet from the extension by clicking "Stop Wallet" in the extension's menu. If you do this, you will need to re-launch the script to use the extension again later.

Getting Started with Linux

Monero Simplewallet Launcher Script

To make launching simplewallet as simple and straightforward as possible, we have created an easy-to-use launcher script.

  • First, open the Terminal and navigate to your main user directory. Then make a new folder called "MoneroWallet".
    To do this in the Terminal, type:

    cd ~/
    mkdir MoneroWallet

  • Next, you will need to have Monero's official simplewallet on your computer. You can get the latest Monero binaries at getmonero.org.
    Unzip the downloaded binaries and copy "simplewallet" into your new MoneroWallet folder.
  • In addition, download "Launch Script - Linux.zip" from the latest release on our GitHub page.
    Unzip the launcher script and copy "MoneroWallet.sh" into your MoneroWallet folder.

Note: It is important that this folder is located at exactly "~/MoneroWallet". To be sure you have set things up correctly, you can type the following command into the Terminal:

cd ~/MoneroWallet && ls

If set up properly, "simplewallet" and "MoneroWallet.sh" should both be returned.

Setting up a new wallet.

The first thing the launcher script does is help set up a new wallet with simplewallet. This has to be done via the command line, but we have tried to make this setup as easy as possible. After this initial setup, you should not need the command line wallet again.

  • In the Terminal, navigate to your new folder. Then run the Monero Simplewallet Launcher script:

    cd ~/MoneroWallet
    ./MoneroWallet.sh

    The script will launch a dialog for creating a new wallet. Follow the dialog, choosing a password and other options. Unless you have your own Monero node running or want to specify a specific node, choose the "remote" option when prompted.
  • Once you have chosen all of your settings, simplewallet will generate a new wallet for you and tell you a secret 25 word "seed". Write down your seed and keep it safe and private. This seed is all you ever need to restore a corrupt or lost wallet.
  • Simplewallet should have told you to type "refresh" to get your wallet going. Do that now.

    refresh

    This will begin to scan Monero's blockchain for your wallet and bring you up to date. This may take some time, especially if you are using a remote node. Once it is done, simplewallet will tell you it is up to date and display a balance.
  • You can now exit the wallet by typing "exit".

    exit

Connecting your extension to simplewallet

Now that you have created a new wallet, the launcher script will work slightly differently. Instead of opening simplewallet in the command line, the script will launch simplewallet in the background on your computer. This allows your wallet extension to talk with simplewallet and send and receive transactions. Any time you want to use your wallet, the script will need to be running.

  • In the Terminal, navigate to your new folder. Then run the Monero Simplewallet Launcher script:

    cd ~/MoneroWallet
    ./MoneroWallet.sh

    This time, the script will ask you to enter the wallet you chose before and pick a node. Then it will launch simplewallet in rpc mode, and give you some feedback. Once finished, you can safely exit the script by typing "exit" or hitting the "x" button in the Terminal window.
  • Now check your chrome extension to see if it is connected online!
    Click the extension icon in the upper right of your browser to open the extension popup.

You can turn off your wallet from the extension by clicking "Stop Wallet" in the extension's menu. If you do this, you will need to re-launch the script to use the extension again later.

If you have never used screen, a quick online search such as "screen command osx" should return plenty of information on how to use it.
